Code to support 16WW Eddi Export Margin analysis at:
https://www.earth.org.uk/eddi-diverter-export-margin-analysis.html

net_energy_2019_2020_2021.xz
is an XZ (LZMA) compressed catenation of all the 2019 2020 2021 Enphase data
used in the model, as a safe separate snapshot.

The Enphase data is published under a CC0 licence.
Model output/results are published under a CC0 licence.

All code elements here are published under an Apache 2.0 licence.
